## What is OWASP Nest?

OWASP Nest is a comprehensive, community-first platform built to enhance collaboration and contribution across the OWASP community. The application serves as a central hub for exploring OWASP projects and ways to contribute to them, empowering contributors to find opportunities that align with their interests and expertise.

## Technology Stack

OWASP Nest is built with modern, open-source technologies:

<Tabs>
  <Tab title="Frontend">
    - **Framework**: Next.js 14 with React 18
    - **Language**: TypeScript
    - **Styling**: Tailwind CSS
    - **Search**: Algolia Search
    - **Data Fetching**: Apollo Client (GraphQL)
  </Tab>
  <Tab title="Backend">
    - **Framework**: Django with Django REST Framework
    - **Language**: Python 3.11+
    - **Database**: PostgreSQL
    - **API**: REST + GraphQL
    - **Search Indexing**: Algolia
    - **AI Integration**: OpenAI
  </Tab>
  <Tab title="Data Sources">
    - **GitHub API**: Project repositories, issues, PRs, releases
    - **OWASP Data**: Projects, chapters, events, committees
    - **Real-time Sync**: Automated data synchronization
  </Tab>
</Tabs>

## Core Data Models

OWASP Nest aggregates and indexes several key entities:

<Info>
  All data is automatically synchronized from authoritative sources including GitHub and OWASP repositories.
</Info>

### Projects

Projects are the heart of OWASP. Each project includes:

- **Types**: Code, Documentation, Tool, Other
- **Levels**: Flagship, Production, Lab, Incubator
- **Metrics**: Stars, forks, contributors, commits, releases
- **Languages & Topics**: Technology stack and focus areas
- **Health Score**: Computed based on activity and compliance

### Chapters

Local OWASP chapters around the world:

- Geographic location with coordinates
- Chapter leaders and members
- Local events and meetups
- Regional information

### Issues & Pull Requests

Contribution opportunities across all OWASP projects:

- Real-time sync from GitHub repositories
- Labels, milestones, and assignees
- AI-generated summaries for complex issues
- Filtered by skill level and topic

### Events

Community gatherings and conferences:

- **Categories**: Global, AppSec Days, Partner events
- Start and end dates with locations
- AI-suggested location data
- Calendar integration
