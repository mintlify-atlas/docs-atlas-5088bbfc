This quickstart guide will help you explore OWASP projects, find contribution opportunities, and connect with the security community.

## Visit OWASP Nest

Navigate to [nest.owasp.org](https://nest.owasp.org) to access the platform.

The home page displays:
- Recent OWASP projects and their activity
- Upcoming community events
- Open issues across projects
- Recent releases and milestones
- Interactive chapters map

## Search for Projects

<Steps>
  <Step title="Access the Search Bar">
    Use the search bar at the top of any page to find projects, issues, chapters, or community members.
  </Step>
  
  <Step title="Try Search Filters">
    Filter projects by:
    - **Level**: Flagship, Production, Lab, Incubator
    - **Stars**: GitHub star count
    - **Activity**: Recent commits and updates
    - **Language**: Programming language
  </Step>
  
  <Step title="Use Structured Search">
    Try advanced queries like:
    ```
    name:security stars:>100
    level:production language:Python
    ```
  </Step>
</Steps>

<Info>
  Learn more about search syntax in the [Search Guide](/guides/using-search).
</Info>

## Explore Project Details

Click on any project to view:

- **Project Overview**: Description, level, and type
- **GitHub Statistics**: Stars, forks, contributors, open issues
- **Health Metrics**: Activity score, maintenance status
- **Open Issues**: Contribution opportunities with labels
- **Recent Releases**: Latest versions and changelogs
- **Project Leaders**: Contact information for maintainers

## Find Issues to Contribute

<Steps>
  <Step title="Browse Issues">
    Navigate to the Issues page or use the search to filter by:
    - `good first issue` - Beginner-friendly tasks
    - `help wanted` - Projects seeking contributors
    - Programming language or technology
  </Step>
  
  <Step title="Review Issue Details">
    Each issue shows:
    - Title and description
    - Associated project and repository
    - Labels indicating difficulty and type
    - GitHub link to view full details
  </Step>
  
  <Step title="Start Contributing">
    Click through to GitHub, read the contributing guidelines, and submit your pull request.
  </Step>
</Steps>

<Tip>
  Check the [Finding Issues Guide](/guides/finding-issues) for strategies to find issues matching your skills.
</Tip>

## Connect with Community

### Join OWASP Slack

1. Visit [owasp.org/slack/invite](https://owasp.org/slack/invite) to join
2. Find the [#project-nest](https://owasp.slack.com/archives/project-nest) channel
3. Use NestBot commands like `/projects`, `/chapters`, `/events`

### Use NestBot Commands

Try these Slack commands:

- `/projects security` - Search for security-related projects
- `/chapters London` - Find chapters near London
- `/events upcoming` - See upcoming OWASP events
- `/ai explain this issue` - Get AI-powered insights
- `/contribute python` - Find Python contribution opportunities

### Find Local Chapters

<Steps>
  <Step title="Open the Chapters Map">
    Navigate to the Chapters page to see the interactive map.
  </Step>
  
  <Step title="Share Your Location">
    Click the location icon to find chapters near you.
  </Step>
  
  <Step title="Connect">
    Click on chapter markers to view details and join their Slack channels.
  </Step>
</Steps>

### Monitor Project Updates
